The Problem

Tangled Git commits complicate the software development lifecycle, making debugging, integration, and code reviews error-prone and time-consuming.

Our Solution: Atommit

Atommit is an automatic developer tool that detects and untangles git commits before committing, partitioning them into atomic commits based on the relatedness of code pieces.

Analyze

Atommit reads code diffs, analyzes syntax trees, and generates dependency graphs.

Partition

It partitions code changes into atomic commits based on the relatedness of the changes.

Suggest

Atommit suggests suitable commit messages and lets users review or modify the results.

How It Works

Atommit provides a workflow that untangles commits by analyzing changes, partitioning them, and suggesting appropriate commit messages.
